# PremiumAuthPlugin

Minecraft Experimental Plugin to add extra security on offline servers

![Platform](https://img.shields.io/badge/platform-PaperMC%20%2F%20Bukkit-blue) ![API](https://img.shields.io/badge/api--version-1.21-green)

## Commands

| Command | Description | Permission |
|---------|-------------|------------|
| `/<command> <4-digit-pin>` | Sets a 4-digit PIN for your account. | `premiumauth.use` |
| `/<command> <4-digit-pin>` | Enters your PIN to verify your identity on login. | `premiumauth.use` |

## How to Use

1. Download the `.jar` from [Releases](https://github.com/Gabrieli2806/PremiumAuth-Plugin/releases).
2. Place it inside your server's `plugins/` folder.
3. Restart or reload the server.
4. Use the commands listed above in-game or from the console.
